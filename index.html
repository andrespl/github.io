Introducción

Este proyecto corresponde al Trabajo Final de Máster de Andrés Puente López. El proyecto consiste en la creación de dos imágenes de Docker para pipelines de metagenómica, estas imágenes contienen lo siguiente:
  1. Un pipeline (Mothur o QIIME) de metagenómica, con las dependencias y paquetes necesarios para ser ejecutado.
  2. Un dataset de prueba

Imagen de Qiime

Esta imagen contiene el pipeline QIIME ( quantitative insights into microbial ecology). Este es un software abierto creado usando PyCogent toolkit para solucionar problemas relacionados con la toma de datos crudos de secuencias. QIIME permite un amplio espectro de análisis de comunidades microbianas y visualizaciones, que han sido fundamentales para estudios recientes, incluyendo análisis de redes, histogramas de diversidad en muestras y análisis donde se comprueba si un cierto grupo central de organismos está representado de forma significativa en ciertos hábitats.

El dataset incluido corresponde a uno de los tutoriales que los desarrolladores de QIIME ofrecen, en este caso para realizar de novo OTU picking y diversos análisis para un amplicon 16S. Estos datos provienen de un estudio realizado por Crawford et al. (2009). Para facilitar el uso y aligerar la carga computacional se ha usado un subset de 5 animales alimentados y 4 en ayunas. Para más información consultar la página de referencia

Imagen de Mothur

Esta imagen contiene el pipeline Mothur. Este se enfoca en ser un software exhaustivo que permita a los usuarios utilizar un solo software para realizar análisis de secuencias de comunidades. Está basado en herramientas previas para ofrecer un software poderoso y flexible para el análisis de secuencias

La información sobre el dataset puede encontrarse aquí.
