Tutorial: como conseguir y utilizar una imagen de Docker

Vamos a ver como poder conseguir cualquiera de las dos imágenes de Docker primero, ya que el proceso es igual para ambas.
  1. Instalar Docker

    2. Es imprescindible contar con un ordenador con los requisitos para instalar Docker. En general estos son únicamente espacio suficiente para la instalación y un sistema operativo compatible. Para realizar la instalación hay que seguir los pasos apropiados para el sistema operativo que se esté utilizando, es recomendable haber revisado antes la página instalación de Docker

  2. Obtener la imagen

    3. Una vez se tiene Docker, hay que obtener la imagen. Para ello habrá que utilizar el comando docker pull andrespl/mothur // docker pull andrespl/qiime o docker run -t -i andrespl/mothur // docker run -t -i andrespl/qiime

    La primer opción solo comprobará si la imágen se encuentra en los archivos locales, y sino es así, la descargará. La segunda, además de lo que haría la primera, la ejecuta. Los comandos -t y -i son necesarios para indicarle a Docker que debe abrir una nueva terminal (pseudo TTY) y permitir la entrada estándar respectivamente. Param más información sobre las posibilidades consultar aquí

  3. Ejecutar los pipelines dentro de la imagen. La información de la ejecución de cada pipelines puede encontrarse en QIIME y en Mothur
